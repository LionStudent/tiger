To run the code, you will need to download matplotlib. You can do this by pasting the code below into the terminal:

cd matplotlib
python -m pip install --upgrade pip --user
python -m pip install -r requirements.txt

To run the program, type the following into the terminal:

python script.py

To load the website page, type the following into the terminal:

python -m http.server 8000

Press "yes", then "Open Link" in the bottom right corner.
If the website doesn't load, copy the URL and paste it into a new tab.
Fin!
